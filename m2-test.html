HOW TO CAPTURE (Path 3 — Static Model Training):

1. Click a letter in the grid below to select it.
2. Hold that ASL handshape in front of the camera.
3. Press Space (or the Capture button) once per snapshot.
4. Aim for 150 captures per sign with variation:
 • Move hand left / right / closer / farther
 • Tilt wrist slightly each shot
 • Don't always hold the same position

HOW TO RECORD A MOTION SEQUENCE:

1. Click a word in the grid to select it.
2. Rest your hand below frame or still.
3. Press R (or the Record button).
4. Immediately perform the sign naturally.
5. Hold still at the end — recording stops automatically.

One press = one sequence = 30 consecutive frames.
Aim for 150 sequences per word.
Each recording takes ~1–3 seconds.

Variation tips:
 • Vary speed (slow, normal, fast)
 • Vary hand position (left/right/closer/far)
 • Don't start from the same position every time

Export saves to data/sequence_data.json — put that
file in your repo's data/ folder.
